# parallax_game
Prototype Unity Game

## Name
Parallax (provisional)

## Concept
Parallax is a 2D side-scrolling roguelite where players pilot a space ship through a shifting, unstable reality.
During each run, gameplay constantly transforms between combat. Adaptation is the core mechanic: mastering the game means learning how to survive when the rules stop being constant.

## Objetive:
Indie roguelite experimental with cards, mutations and game modes
